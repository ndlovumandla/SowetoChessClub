# Soweto Chess Club

## Storyline
Every Saturday morning in a cramped community hall in Soweto, retired mathematics teacher Mr. Dlamini runs a free chess club for children aged 8 to 17. He has no computer, no app — just a handwritten register, torn scoresheets, and a cardboard rating board on the wall. After a local NGO donates a laptop, his granddaughter Zanele — a first-year computer science student — promises to build him a simple program to track players, their ratings, wins, losses, and weekly match results. She starts from scratch, modelling each player as an object, the club as a class, and every Saturday session as a recorded event with outcomes.

## How to Run the Project

```bash
# 1. Clone the repository
git clone https://github.com/ndlovumandla/SowetoChessClub.git
cd SowetoChessClub

# 2. Compile the Java file
javac SowetoChessClub.java

# 3. Run the program
java SowetoChessClub
```

## Features
- Player management with ratings and match history
- Chess Club class managing multiple players
- Realistic simulation of Saturday matches
- Full OOP demonstration with encapsulation

## Learning Objectives
- Classes and Objects
- Constructors
- Encapsulation using private fields
- Getters and Setters
- Interacting classes in a real-world scenario
